Python基础课程

掌握Python基础编程素养，包括语法、数据结构、函数、面向对象和文件操作

1. Python基础语法

Python的语法简洁明了，以下是一些基础概念：

1.1 注释

# 单行注释 """ 多行注释 可以跨越多行 """

1.2 变量

# 变量赋值 name = "Python" age = 30 pi = 3.14

1.3 数据类型

  • 整数 (int): 如 1, 2, 3
  • 浮点数 (float): 如 3.14, 2.5
  • 字符串 (str): 如 "Hello", 'Python'
  • 布尔值 (bool): True, False
  • 列表 (list): 如 [1, 2, 3]
  • 元组 (tuple): 如 (1, 2, 3)
  • 字典 (dict): 如 {"name": "Python", "age": 30}
  • 集合 (set): 如 {1, 2, 3}

1.4 运算符

  • 算术运算符: +, -, *, /, //, %, **
  • 比较运算符: ==, !=, >, <, >=, <=
  • 逻辑运算符: and, or, not
  • 赋值运算符: =, +=, -=, *=, /=

1.5 控制流

# if-else语句 if condition: # 执行代码 elif another_condition: # 执行代码 else: # 执行代码 # for循环 for item in iterable: # 执行代码 # while循环 while condition: # 执行代码

2. 数据结构

2.1 列表 (List)

# 创建列表 numbers = [1, 2, 3, 4, 5] # 访问元素 first = numbers[0] # 修改元素 numbers[0] = 10 # 添加元素 numbers.append(6) # 切片 sublist = numbers[1:4]

2.2 元组 (Tuple)

# 创建元组 coordinates = (10, 20) # 访问元素 x = coordinates[0] # 元组不可修改 # coordinates[0] = 30 # 会报错

2.3 字典 (Dictionary)

# 创建字典 person = {"name": "Alice", "age": 25, "city": "New York"} # 访问值 name = person["name"] # 添加/修改键值对 person["email"] = "alice@example.com" # 遍历字典 for key, value in person.items(): print(key, value)

2.4 集合 (Set)

# 创建集合 fruits = {"apple", "banana", "orange"} # 添加元素 fruits.add("grape") # 移除元素 fruits.remove("banana") # 集合运算 set1 = {1, 2, 3} set2 = {3, 4, 5} union = set1 | set2 # 并集 intersection = set1 & set2 # 交集

3. 函数与模块

3.1 函数定义

def greet(name): """函数文档字符串""" return f"Hello, {name}!" # 调用函数 message = greet("Python") print(message)

3.2 函数参数

# 位置参数 def add(a, b): return a + b # 默认参数 def greet(name, greeting="Hello"): return f"{greeting}, {name}!" # 关键字参数 result = add(a=5, b=3)

3.3 模块

# 导入模块 import math print(math.pi) # 导入特定函数 from math import sqrt print(sqrt(16)) # 导入所有函数 from math import *

3.4 创建自己的模块

创建一个名为 mymodule.py 的文件，然后在其他文件中导入：

# mymodule.py def say_hello(): return "Hello from my module!" # 在其他文件中 import mymodule print(mymodule.say_hello())

4. 面向对象编程

4.1 类的定义

class Person: def __init__(self, name, age): self.name = name self.age = age def greet(self): return f"Hello, my name is {self.name}"

4.2 创建对象

# 创建Person对象 person = Person("Alice", 25) # 访问属性 print(person.name) # 调用方法 print(person.greet())

4.3 继承

class Student(Person): def __init__(self, name, age, student_id): super().__init__(name, age) self.student_id = student_id def study(self): return f"{self.name} is studying" # 创建Student对象 student = Student("Bob", 20, "S123") print(student.greet()) # 继承自Person print(student.study()) # Student特有方法

5. 文件操作与异常处理

5.1 文件操作

# 写入文件 with open("example.txt", "w") as f: f.write("Hello, Python!\n") f.write("This is a test file.") # 读取文件 with open("example.txt", "r") as f: content = f.read() print(content) # 读取文件行 with open("example.txt", "r") as f: lines = f.readlines() for line in lines: print(line.strip())

5.2 异常处理

try: # 可能引发异常的代码 result = 10 / 0 except ZeroDivisionError: # 处理特定异常 print("除数不能为零") except Exception as e: # 处理其他异常 print(f"发生错误: {e}") else: # 没有异常时执行 print("计算成功") finally: # 无论是否有异常都会执行 print("操作完成")
