第1课：Python简介与环境搭建

1. Python简介

Python是一种高级编程语言，由Guido van Rossum于1989年创建。它以简洁、易读的语法而闻名，是初学者的理想选择，同时也被许多专业开发人员用于各种应用场景。

2. Python的特点

  • 简单易学：语法简洁明了，代码可读性高
  • 面向对象：支持面向对象编程范式
  • 解释型：无需编译，直接运行
  • 跨平台：可在Windows、Mac、Linux等多种操作系统上运行
  • 丰富的库：拥有大量内置和第三方库

3. 安装Python

访问 Python官方网站 下载并安装适合你操作系统的Python版本。

4. 验证安装

安装完成后，打开命令行或终端，输入以下命令验证Python是否安装成功：

python --version # 或 python3 --version (在某些系统上)

5. 运行第一个Python程序

创建一个名为 hello.py 的文件，输入以下代码：

print("Hello, World!")

然后在命令行中运行：

python hello.py

练习
